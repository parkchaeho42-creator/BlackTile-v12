# BlackTile V12

> Binance BTCUSDT Perpetual Futures Automated Trading System

## Overview

BlackTile V12는 Binance USDT-M BTCUSDT Perpetual 전용 자동매매 시스템입니다.

이 프로젝트는 하나의 거래 엔진으로 다음 모드를 지원하도록 설계됩니다.

- Backtest
- Paper Trading
- Shadow Live
- Live Trading

## Development Goals

- 안정적인 자동매매 엔진 구축
- 동일 엔진 기반 운영
- AI 기반 보조 의사결정
- Mini PC 기반 24시간 운영
